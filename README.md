# Channel Coding Demo

An interactive standalone web app for learning core concepts of channel coding with visual and animated examples.

## About this project

This project explains and demonstrates how linear block codes work using:

- generator matrices (`G`)
- parity-check matrices (`H`)
- syndrome computation (`s = r * H^T`)
- matrix multiplication animations step by step
- interactive parity definition and custom encoding
- noise simulation and error detection/correction intuition
- standard array construction with coset leaders and syndromes

## Run locally

Open `index.html` directly in your browser.
